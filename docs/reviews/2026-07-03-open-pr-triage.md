- canonical 계열 테이블(`contributors`, `contribution_events`, `contributor_points`, `source_candidates`)은 **프로덕션 DB에 존재하지 않는다**. 해당 테이블에 쓰던 코드가 report 흐름의 상시 500 에러 원인이었고, #419가 이를 실제로 동작하는 `awardPoints() → contributor_point_events` 경로로 통일했다.
- 따라서 게임화/포인트 관련 PR은 `contributor_point_events` 모델과의 정합성이 1차 판단 기준이다.
